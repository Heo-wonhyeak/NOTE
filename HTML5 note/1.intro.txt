1. UI (User Interface)
    1) 인터페이스 : 정보를 교환하는 공유 경계
        - 로그인 기능
        - 검색 기능
        - 체크 박스

2. API (Application Programming Interface)
    1) 컴퓨터 시스템끼리 정보를 교환하는 공유 경계임.
        - 서버 주소/A(로그인 기능)
        - 서버 주소/B(회원가입 기능)
        - 서버 주소/C(글쓰기 기능)
        - 서버 주소/D(불러오기 기능)

3. Front end 개발
    1) HTML, CSS, JS를 사용해 데이터를 그래픽 사용자 인터페이스(GUI)로 변환하고
       그것으로 사용자와 상호작용할 수 있도록 하는 것.
    2) HTML(Hyper Texgt Markup Language)
            - 페이지의 제목, 문단, 표, 이미지, 동영상 등 웹의 구조를 담당
       HTML + CSS(Cascading Style Sheets)
            - 실제 화면에 표시되는 방법(색상, 크기, 폰트, 레이아웃)등 지정해 콘텐츠를 꾸며주는 시각적인 표현(정적) 담당.
       HTML + CSS + JS(JavaScript)
            - 콘텐츠를 바꾸고 움직이는 등 페이지를 동작시키는 동적 처리를 담당.

4. 웹(Web)
    1) 인터넷에 연결된 사용자들이 서로 정보를 공유할 수 있는 서비스임.
    2) HTML은 웹 서비스에 사용되는 마크업 언어임.
        - 마크업 : 문서나 데이터 구조를 태그(tag)등을 이용하여 명시하는 언어임.
        - 웹 페이지에 표시(mark)를 하는 언어 
    3) 웹 브라우저에 보이는 화면
        - 텍스트, 이미지, 동영상, 오디오 같은 요소를 HTML로 구조화해 담은 문서를 해석해서 보여준 결과임.
    4) 네트워크 : 컴퓨터와 컴퓨털르 연결해주는 망.
        - 인터넷 : 네트워크가 모여진 망.
    5) 웹의 통신규약 : HTTP/HTTPS
        - 요청과 응답이 일어날 때 웹 페이지 코드가 서버에서 클라이언트로 전달되는데, 이때 HTTP 통신 규약을 이용함

5. 웹페이지
    1) 정보를 하이퍼텍스트라는 특별한 양식으로 제공하는 웹 문서임.
    2) 하이퍼텍스트
        - 참조를 통해 클라이언트가 한 문서에서 다른 문서로 즉시 접근할수 있는 텍스트임.
    3) 브라우저
        - 하이퍼텍스트를 해석해 보여주는 대표적인 클라이언트임.

6. 웹 표준 (Web Standard)
    1) 웹에서 사용되는 표준 기술이나 규칙 
    2) W3C(World Wide Web Consortium)의 표준화 제정 단계의 '권고안(REC)'에 해당하는 기술.
    3) 1991년 HTML 1.0
       1995년 HTML 2.0
                                        1996년 CSS 1            JavaScript ES1
       1997년 HTML 3.2, 4.0
                                        1998년 CSS 2            JavaScript ES2
       1999년 HTML 4.01                                         JavaScript ES3
                                        2005년 CSS 3
       2007년 HTML 5
                                                                2009년 이후 JavaScript ES5(오늘날의 주된 업데이트 내용)
    
7. 크로스 브라우징(Cross Browsing)
    - 조금 다르게 구동되는 여러 브라우저에서, 동일한 사용자 경험(같은화면, 같은 동작)을 줄수 있도록 제작하는 기술, 방법.
        - ex) Chrome/Firefox/Edge/Safari/Opera/Swing etc...

8. 특수 문자 용어
    - 백틱(Backtick), 그레이브(Grave) : `
    - 틸드(Tilde), 물결 표시 : ~
    - 앳(At sign), 골뱅이 : @
    - 캐럿(Caret) : ^
    - 대시(dash) : -
    - 언더스코어(Underscore), 언더바 : _ 
    - 콜론(Colon) : :
    - 세미콜론(Semi Colon) : ;
    - 백슬래쉬(Backslash) : \
    - 퍼랜서시스(Parentheses), 소괄호 : ()
    - 브레이스(Brace), 중괄호 : {}
    - 스퀘어 브래킷(Square Brackets), 대괄호 : []
    - 앵글 브래킷(Angle Brackets), 꺽쇠괄호 : <>
