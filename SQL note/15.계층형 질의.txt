1. 오라클 계층형 질의 (hierarchical query)
    1) 테이블에 계층형 데이터가 존재하는 경우 데이터를 조회하기 위해서 계층 형 질의를 사용
    2) 계층형 데이터란 동일 테이블에 계층적으로 상위와 하위 데이터가 포함된 데이터를 말함.
        - START WITH 조건
            - 계층 구조 전개의 시작 위치를 지정하는 구문
            - 루트 데이터를 지정함
        - CONNECT BY [PRIOR]
            - 다음에 전개될 자식 데이터를 지정하는 구문
    3) 가상 컬럼
        - LEVEL 1 : 루트데이터
        - LEVEL 2 : 바로 하위 데이터
            - 하위데이터가 있을 때마다 1씩 증가
        - CONNECT_BY_ISLEAF
            - 전개과정에서 해당 데이터가 LEAF 데이터면 1 , 그렇지 않으면 0
                -LEAF : 잎사귀 즉 마지막 데이터!(최하위)
