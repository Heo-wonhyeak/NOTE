1. 안드로이드 특징
    1) 핵심 커널 (kernel)
        - 리눅스
    2) 애플리케이션 개발 언어
        - 자바, 코틀린(자바기반 - 자바 100% 활용가능)
    3) 오픈 소스 지향
    4) 응용 프로그램 프레임워크 (Application Framework)
        - 안드로이드 API가 존재한 곳 
        - 안드로이드 하드웨어에 접근할 때 API를 통해서만 가능함

2. 안드로이드 구조
    1) 안드로이드 런타임
        - 자바 코어 라이브러리
        - 안드로이드 런타임(ART)

3. Hello Android App
    1) Name : 프로젝트 이름
    2) Minimum SDK : 앱이 작동하는 최하 버전

4. 안드로이드 프로젝트 생성
    1) 2개 항목
        - layout Name : 기본 화면인 XML 파일 이름으로 지정
        - Activity Name : 기본소스인 Java 파일 이름으로 지정
