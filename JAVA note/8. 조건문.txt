1. 조건문이란?
  1) 주어진 조건에 따라 다른 실행이 이루어지도록 구현
  2) 조건문은 조건식의 결과가 'true인 경우', 'false인 경우'의 두가지 흐름을 만들어냄
  3) if문, switch문 

2. if문 문법
  1) if(조건식) {
      수행문;       // 조건식 '참'인 경우 수행문이 실행
  }
  2) if ~ else문
    if (조건식) {
        수행문1;             //조건식 '참'인 경우 수행됨
    }
    else {
        수행문2;              //조건식 '참'이 아닌 경우 수행됨
    }
  3) 조건이 여러개 일때의 if문 
    if ~ else if ~ else ... 문
    - 하나의 상황에 대한 조건이 여러개로 나뉘고
      각 조건에 다른 수행이 이루어져야 할 경우 사용
    - 각 조건은 상호 배타적임
     if (조건식1) {
        수행문1;             //조건식1 '참'인 경우 수행 후 전체 조건문을 빠져나감
     }
     else if(조건식2){
         수행문2;            //조건식2 '참'인 경우 수행 후 전체 조건문을 빠져나감
     }
      else if(조건식2){
         수행문3;            //조건식3 '참'인 경우 수행 후 전체 조건문을 빠져나감
     }
      else {
         수행문4;            //모두 거짓인 경우 수행 후 전체 조건문을 빠져나감 (defult)
     }

     수행문5; // 조건문 과 별도로 무조건 실행
  4) 중첩 if문
    - if문 안에 또 다른 if문 중첩해서 넣을 수 있음.
    - 중첩 if문 개수가 제한이 없음(but, 너무 많이 중첩하면 가독성이 저해됨.)
