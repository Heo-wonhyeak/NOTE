1. 자바 소개
   1) 1991년 썬마이크로시스템즈에 제임스 고슬링(자바의 아버지)의 팀에서 시작됨.
   2) C/C++ 언어(특징:포인터 - 메모리 주소가 포함[장점이자 단점])가 주류였는데, 다양한 플랫폼마다 매번 다른 기계어로 컴파일을 해야함.
   3) 플랫폼 독립적인 기능이 필요했음.
   4) 1996년 JDK(Java Development kit) 1.0 공개됨(처음엔open source 는 아니었음)
      - JDK 1.1(1997)
      - J2SE(Java 2 Platform Standard Edition) 1.2
      - J2SE 1.3
      - J2SE 1.4
      - J2SE 5.0
      - JAVA SE 6
      - JAVA SE 8 : LTS version 도입 (Long Term Supports) [10년 기술지원 2025년까지]
      ~~~
      - JAVA SE 17
   5) 2010년 1월 오라클에 인수됨 (open source가 됨으로 상업적인 문제로 인해)

2. 자바의 특징
   1) 고급 프로그래밍 언어임.
   2) 객체 지향 언어임(object oriented programming)임.
   3) 모든 운영체제에서 실행 가능함.
   4) 메모리 관리를 자동으로 해줌. (그때 당시 굉장한 차별점)
   5) 무료 라이브러리가 풍부하게 제공됨.
   6) 동시 작업을 할 수 있는 멀티 쓰레드 기능 제공함.

3. 자바의 기술 종류
   1) Java SE (Java Standard Edition)
     - 데스크탑 컴퓨터의 응용 프로그램 개발용 자바 기술 , 안드로이드(java se의 일부를 사용하고있음) [2008년]
   2) Java EE (Java Enterprize Editon)
     - 서버용 응용 프로그램 개발용 자바 기술
     - servlet, JSP , Spring Framework 등
   3) Java ME (Java Micro Edition)

4. 자바의 기본 도구
   1) javac.exe : 자바 컴파일러 (Java Compiler)
     - 컴파일 형식
        javac 파일명.java
   2) java.exe : 자바 런처 (Java Launcher)
     - 실행 형식
        java 파일명

5. 컴퓨터 진수 입력 방식
  1) 10진법 = 실생활에 사용하는 표기
  2) 2진법 = 숫자 앞에 0b(B)를 추가 표기
  3) 8진법 = 숫자 앞에 0을 추가 표기
  4) 16진법 = 숫자 앞에 0x(X) 표기 + [10=A 11=B ... 15=X]
